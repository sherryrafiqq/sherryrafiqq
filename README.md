⚡ about me
I'm a final-year Data Science & Computer Science student specializing in AI Engineering — I build systems that sit between raw data and useful language: RAG pipelines, LLM application layers, and the backend infrastructure that keeps them fast under real load.
I care more about whether it actually works under real constraints than whether it demos well once.
<br/>
🪞 what I'm building right now
AntiDoom — a "reality check" for the news you scroll past
News headlines are optimized for alarm, not accuracy. AntiDoom takes any article URL and returns two things a headline never gives you:
📊 Reality Check — statistical & historical context, so a scary number lands with actual scale attached
🌱 Hope Signals — real human responses and emerging solutions already in motion, pulled from search and ranked for relevance
Stack: Python · LangChain · FAISS · FastAPI · LLM prompt chains
> Currently mid-build on the ranking layer — extraction, LLM reasoning, and search modules are done and tested.
